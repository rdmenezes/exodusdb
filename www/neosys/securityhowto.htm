Before you can use NEOSYS.NET via a mapped drive or shared directory you must change your security settings once per workstation.

This procedure is not necessary when accessing NEOSYS via a web server.

  • In the Internet Explorer menu (above), choose Tools, Internet Options
  • Click the Security tab
  • Click the Trusted Sites icon
  • Click the Sites button
  • Clear (uncheck) the Require Server Verification checkbox
  • In the text box "Add this Web Site", enter something like "file://XXXXXXXX" without the quotes but put your own server name (eg file://NT_SERVER) then click Add. You can find your server name at the top of the screen when starting NEOSYS.
  • Click OK
  • Click the Custom Level button
  • Enable "Initialise and Script ActiveX Controls Not Marked as Safe"
  • Click OK or press Enter to close Security Settings
  • Click OK or press Enter to close Internet Options
  • Click or press Ctrl+Backspace to return to the NEOSYS login page.
